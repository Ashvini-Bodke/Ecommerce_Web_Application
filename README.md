
# 🛒 Ecommerce Web Application

A full-stack Ecommerce Web Application built to provide a seamless online shopping experience. The application enables users to browse products, manage products, and interact with the system through a responsive and user-friendly interface.

## 🚀 Tech Stack

### 🔹 Frontend

* React.js
* JavaScript
* HTML5
* CSS3
* Bootstrap

### 🔹 Backend

* Spring Boot
* Java
* REST APIs

### 🔹 Database

* MySQL

## ✨ Features

* 👤 User Authentication
* 🛍️ Product Browsing
* 📦 Product Management
* 🔍 Product Search
* 🔗 REST API Integration
* 📱 Responsive User Interface

## 🏗️ Project Architecture

* Frontend (React.js) communicates with Backend through REST APIs.
* Backend developed using Spring Boot handles business logic and data processing.
* MySQL database is used for storing application data.
* Client-server architecture ensures scalability and maintainability.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Ashvini-Bodke/Ecommerce_Web_Application.git
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

### Backend Setup

```bash
cd backend
mvn spring-boot:run
```
